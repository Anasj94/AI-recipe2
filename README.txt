### Project Overview:

The AI Recipe Generator is a forward-thinking application of artificial intelligence (AI) in the culinary arts.
The system utilizes advanced machine learning algorithms to analyze huge datasets of recipes, ingredients,
and techniques. Understanding and interpreting user input in plain language, it permits frictionless
communication between user and system.

This model is built on feedback from users, and constantly adjusts its recommendations in response to
their likes and dislikes; the result is an eternally changing eating experience. By utilizing deep learning
techniques, the system steers toward habit and away from temptation. So recipes can be made that are
healthy and thus palate-pleasing for you. The AI Recipe Generator has the potential to greatly benefit
professional chefs and establishments involved in the food business as it enables menu planning, specially
branded dishes, and following trends. But introducing AI into the culinary domain raises many ethical issues
such as cultural appropriation, authenticity, and what it means for traditional culinary practices.

### Instructions for Setup and Execution:

-Add a `.env` file to the repository root directory.
-In the `.env` file, add the following environment variables:

```
OPENAI_API_KEY=""
TOKENIZERS_PARALLELISM=false
```
-Replace `OPENAI_API_KEY` with your OpenAI API key.
-Enter the following command in your terminal to start the application:

```bash
streamlit run app.py
```
